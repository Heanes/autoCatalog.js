autoCatalog.js 文档说明

配置参数说明

# 配置项 类型 默认值 说明
1 outlineSelector array ['h2', 'h3', 'h4'] 大纲选择器
2 catalogContainer dom || jQuery dom undefined 放置生成目录的容器dom
3 autoCounter boolean false 是否自动为大纲计数(即自动生成编号)
4 showChapterNumber boolean false 是否显示章节编号
5 showChapterSymbol boolean false 是否显示章节符号
6 formatChapterAnchor(chapter) function undefined

章节锚链接自定义格式化

参数：chapter 章节对象，具体值可以调试查看

默认效果是形如"1.1 第一章第一节"这种效果
7 changeUrlAnchor boolean false 是否更改页面URL的锚链接。目前许多网页应用使用#做页面路由，所以增加此选项。
8 scrollContentAnimationDelay number 400 滚动内容的动画延时，为0时没有动画效果
9 scrollToContentOffset number 0 点击目录时，页面跳转到对应章节处的偏移值
10 useButtonToScroll boolean false 是否使用按钮来滚动目录
11 scrollCatalogAnimationDelay number 400 目录滚动动画延时，为0时没有动画效果
12 clickToScrollCatalogStep number 48 当目录较长时，右侧会出现使目录上下滚动的按钮，此值定义点击一次按钮滚动导航目录移动的高度
13 alwaysShow boolean false 是否一直显示目录。若为false，则当文章内容不在页面视窗范围中时，目录自动隐藏。
14 collapseOnInit boolean false 初始化时是否折叠显示。true - 折叠显示，false正常显示
